
## CRYPTOGRAPHIC OPERATIONS

| **FILE_NAME** | **DESCRIPTION** |
| --- | --- |

| EncryptUsing_AESGCM_FIPS_Mode.java | Demonstrates how to use AES-GCM encryption mechanism in FIPS mode. |
